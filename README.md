# CS50028-ComputerVision

复旦大学 **CS50028 计算机视觉**课程代码仓库。

## 仓库结构

- **`HW1/`** — 第一次作业：从零开始构建三层神经网络分类器，实现 Fashion-MNIST 图像分类（具体依赖、运行方式见 **`HW1/README.md`**）。
- **`HW2/`** — 期中作业：宠物图像分类（ResNet 微调与注意力）、VisDrone 无人机场景检测与视频多目标跟踪、Oxford-IIIT Pet 三分类语义分割（U-Net + CE/Dice 损失对比）。包含三个子任务目录 `task1_classification/`、`task2_detection_tracking/`、`task3_segmentation/`。数据集与训练输出等大文件见 [Google Drive](https://drive.google.com/drive/folders/1cgVH-8hv9I9M-6XvFUm0BxcQIiJ1wesb?usp=drive_link)，详细说明见 **`HW2/README.md`** 与 **`HW2/README_GDrive.md`**。
